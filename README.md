# FitHer — 女性健身平台

专为女性打造的一站式健身平台。个性化训练计划、科学饮食建议、动作视频指导，帮你高效达成健身目标。

## 快速使用

**下载 `dist/index.html`，双击浏览器直接打开，无需安装任何东西。**

## 功能

- **智能训练计划**：根据目标（减脂/增肌/塑形）、场景（健身房/居家）、时长（15-90分钟）自动生成专属计划
- **动作指导**：92 个器械+跟练视频，B站嵌入播放，一键跳转
- **饮食建议**：分减脂/增肌/保持三期饮食原则 + 食物库 + 热量参考
- **经期适配**：自动识别经期，调整训练强度
- **打卡追踪**：记录每天训练完成情况

## 技术栈

React 19 + Vite 8 + Tailwind CSS v4 + Framer Motion 12

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
# 产出 dist/index.html
```
